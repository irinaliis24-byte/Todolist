## TodoList

This project is a **Trello-inspired TodoList application** built with React, TypeScript, and Vite as a learning project. The goal of the application is to practice building modern frontend applications using React, focusing on component-based architecture, state management, and interactive user interfaces.

The app allows users to organize their tasks in a visual way using **lists and cards**. Users can create task lists, add tasks to them, and manage their workflow in a simple and structured interface.

### Features

* Create and manage **multiple todo lists**
* Add, edit, and delete tasks
* Organize tasks inside lists
* Dynamic UI built with **React components**
* Responsive and user-friendly interface

### Technologies

* **React**
* **TypeScript**
* **Vite**
* **SCSS**

### Purpose of the Project

The purpose of this project is to practice modern frontend development and improve skills in:

* Building applications with React
* Using TypeScript for safer and more maintainable code
* Setting up projects with Vite
* Creating reusable UI components
* Managing application state and events
* Structuring scalable frontend projects

### Future Improvements

* Drag and drop tasks between lists
* Task labels and priorities
* Data persistence (LocalStorage or backend API)
* User authentication

---

This project demonstrates the core ideas behind **task management tools like Trello**, while keeping the implementation simple and focused on learning React.
